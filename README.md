HUVVSM — To-Do App

TaskFlow is a modern, minimal, and responsive To-Do application built with vanilla HTML, CSS, and JavaScript.
It features a glassmorphism UI, HUVVSM gradient identity, and localStorage persistence.

⸻

Overview

TaskFlow allows users to:
	•	Add tasks
	•	Mark tasks as completed
	•	Edit tasks inline
	•	Delete tasks
	•	Filter tasks (All / Todo / Done)
	•	Clear completed tasks
	•	Persist data using localStorage
	•	Use the app on both desktop and mobile devices

The design focuses on clarity, simplicity, and smooth visual experience.

⸻

Architecture
	•	index.html → Structure & semantic layout
	•	style.css → UI system, glassmorphism, aurora background, responsiveness
	•	script.js → Application logic, state management, localStorage handling

⸻

Features

Task Management
	•	Create new tasks
	•	Toggle completion status
	•	Inline editing without browser prompts
	•	Delete individual tasks
	•	Clear all completed tasks

Filtering System
	•	View all tasks
	•	View only pending tasks
	•	View only completed tasks

Persistence
	•	Tasks are saved in localStorage
	•	Data remains after page reload

UI & Design
	•	Dark glassmorphism card layout
	•	HUVVSM cyan/violet gradient identity
	•	Animated aurora background
	•	Gradient action button
	•	Responsive design for mobile screens

⸻

Technologies Used
	•	HTML5
	•	CSS3 (Flexbox, Grid, gradients, animations)
	•	JavaScript (DOM manipulation, localStorage)

No frameworks. No libraries. Pure front-end implementation.

⸻

Responsive Design

The application adapts to smaller screens using media queries:
	•	Buttons become touch-friendly
	•	Layout stacks vertically
	•	Card resizes to fit viewport
	•	Mobile spacing adjustments

⸻

How to Run
	1.	Clone the repository
	2.	Open index.html in your browser
or
	3.	Use Live Server in VS Code

No build process required.
